# Monster sprites

This directory is for local monster sprite GIFs used by Zenymob2.

Expected file names match Hercules/nn.ai4rei sprite names, for example:

```text
SLEEPER.gif
GOAT.gif
MANDRAGORA.gif
GOLDEN_BUG.gif
```

Run this from the repository root to download sprites for the monsters currently present in `public/data/monsters.json`:

```bash
npm run sync:sprites
```
